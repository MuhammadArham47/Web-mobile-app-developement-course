HTML5: Figure and Figure Caption

<figure></figure>

Images often come withcaptions. HTML5 has introduced a new element to contin images and their and their caption so that the two are associated.

You can have more than one image inside the element as long as they all share the same caption.

<figcaption></figcaption>

The element has been added to HTML5 in orderd to allow web page authors to add a caption to an image.

Before these elements were created there was no way to associated an element with its caption.

Older browser that do not understand HTML5 elements simply ignore the new elements and display the content of them.
